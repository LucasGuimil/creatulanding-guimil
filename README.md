# Tienda AV 
Este proyecto se basa en una tienda de ropa para activistas por los derechos de los animales de la organización Anonymous for the Voiceless, el mismo se encuentra basado en los productos que se encuentran en la [tienda oficial](https://www.anonymousforthevoiceless.org/shop). 

### Tecnologías utilizadas para la creación de este proyecto:
- React + Vite
- Firebase

### Herramientas utilizadas para el diseño visual:
- Bootstrap v5.0 (CDN)
- Sweet Alert 2 v11.21.0

## Guía de instalación
Se requiera instalar las dependencias a través del comando `npm install` e inicializar el proyecto con el comando `npm run dev`.
